Syntax: TRUjQ(selector).actionorevent()

Selectors are document, string for selection - HTML element type, id, class, no other complex selectors (not difficult to implement though) , this as a DOM object, i.e., HTMLElement object, HTMLElement object

Return values from TRUjQ(selector): Three different TRUjQ objects. 1. TRUjQ_document - from TRUjQ(document) 2. TRUjQ_htmlselector - from TRUjQ('...') 3. TRUjQ_htmlelement - from TRUjQ(this) or TRUjQ(HTMLElement)

Event methods supported in TRUjQ objects 1. ready() for document (actually load on window) 2. click() 3. dblclick()

Action methods supported in TRUjQ objects: 1. css() 2. attr() 3. html() 4. hide() 5. show() 6. toggle()

Properties in TRUjQ. 1. version - E.g., TRUjQ.version
